TUT 3 - 300 WORDS AND BLOG WIREFRAMES

Git benefits each designer, organisation and team differently, it is one of the greatest current version control systems. Git is a distributed version-control system used for code, it tracks and saves snapshots of your entire codebase and allows you to easily deliver it to other developers. This means that every designer or team member working on the project has access to the central repository and can have their own localised repository. If your design team is using Git it would benefit your team a lot as you wouldn’t need to train new team members as they have possibly previously used the system before. Another server linked to Git is Github, this system hosts your code on a distant Git server with a web interface. It permits you to co-operate with other designers on the same server and it also comes as an app which can be used on your personal laptops at the comfort of your home. Designers that want to create a website or want to redesign a webpage should use Git as it has a feature of branches that allow you to rapidly prototype websites. If a designer was to make a new user flow or wanted to replace images or icons, making a new branch gives them an entire new working place to rearrange and change everything around. This lets the designers see all the different changes they have made on the different branches. Overall, Git is a fast and reliable system that works well with all the different sections in a business. Groups of designers should use Git for creating and sharing their websites as it will benefit their overall outcome.
